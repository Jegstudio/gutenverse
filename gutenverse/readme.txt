== Description ==

Gutenverse is a WordPress Gutenberg blocks plugin and Site Editor page builder for creating fast, responsive websites without code. It includes 59 WordPress blocks, 600+ templates, global styles, responsive controls, popup builder tools, and block theme support.

Use Gutenverse as a lightweight WordPress page builder for the native Block Editor. Build landing pages, business websites, blogs, portfolios, headers, footers, archive templates, single post templates, popups, and full site designs with Gutenberg blocks.

Instead of replacing WordPress with a separate visual builder, Gutenverse extends Gutenberg and the WordPress Site Editor. It is built for block themes, Full Site Editing (FSE), and modern WordPress websites.

== Installation ==

= From The WordPress Plugin Dashboard =

1. Go to Plugins > Add New in your WordPress dashboard.
2. Search for "Gutenverse".
3. Click Install Now.
4. Click Activate.
5. Open the WordPress Block Editor or Site Editor and search for Gutenverse blocks.

= Using The Upload Method =

1. Download the Gutenverse plugin ZIP file.
2. Go to Plugins > Add New in your WordPress dashboard.
3. Click Upload Plugin.
4. Choose the Gutenverse ZIP file.
5. Click Install Now.
6. Activate the plugin.

For more help, read the [Gutenverse installation guide](https://gutenverse.com/docs/plugin-installation/).

== Frequently Asked Questions ==

= Does Gutenverse require the Gutenberg plugin? =

No. WordPress 5.9 and newer already include the block editor features needed by Gutenverse. You do not need to install the separate Gutenberg plugin unless you specifically want to test experimental Gutenberg features.

= Is Gutenverse a Gutenberg blocks plugin? =

Yes. Gutenverse is a Gutenberg blocks plugin for the WordPress Block Editor. It adds layout blocks, content blocks, design blocks, post blocks, popup tools, and template building features for WordPress websites.

= Is Gutenverse a WordPress page builder? =

Yes. Gutenverse works as a WordPress page builder built on top of Gutenberg and the native Site Editor. You can build pages, landing pages, headers, footers, archive templates, single post templates, and full site layouts without using a separate page builder interface.

= Does Gutenverse work with the WordPress Site Editor? =

Yes. Gutenverse is built for the WordPress Block Editor, Site Editor, block themes, and Full Site Editing workflows. You can use it to create pages, templates, headers, footers, archive layouts, and single post layouts.

= What theme should I use with Gutenverse? =

Gutenverse works best with block themes that support the WordPress Site Editor. You can use the free [Unibiz block theme](https://wordpress.org/themes/unibiz/) or other block themes that follow WordPress Full Site Editing standards.

= Can I use Gutenverse with other Gutenberg block plugins? =

Yes. Gutenverse is designed to work inside the native WordPress Block Editor and should work alongside most well-coded Gutenberg block plugins. Compatibility can vary depending on how other plugins register blocks, styles, and editor scripts.

= How do I find Gutenverse blocks in the editor? =

Open the block inserter in the WordPress editor and search for "Gutenverse". You can also search for a specific block, such as "Gutenverse Accordion", "Gutenverse Gallery", or "Gutenverse Button".

= What should I do if a block shows unexpected or invalid content? =

After some updates, WordPress may show an "Attempt Block Recovery" button for blocks affected by older markup or styling changes. Click that button first. If the issue continues, please open a support ticket and our team will help you resolve it.

= Is Gutenverse lightweight? =

Gutenverse is built for the native WordPress Block Editor and focuses on a fast editing and frontend experience. You can also disable unused Gutenverse blocks with block management tools to keep your workflow cleaner.

= Does Gutenverse collect data? =

Gutenverse includes an optional newsletter form in the dashboard. If you subscribe, the email address you manually submit is sent to gutenverse.com so we can send Gutenverse updates. No newsletter subscription data is sent automatically without your consent.

For more details, read the [Gutenverse terms and conditions](https://gutenverse.com/terms-and-conditions/).
